## 🔄 State Management Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                      USER INTERACTION                            │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    OPTIMISTIC UPDATE                             │
│  Update UI immediately (setData)                                 │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                       API CALL                                   │
│  await api.performAction(payload)                                │
└─────────────────────────────────────────────────────────────────┘
                              │
                ┌─────────────┴─────────────┐
                ▼                           ▼
        ┌──────────────┐          ┌──────────────┐
        │   SUCCESS    │          │    FAILURE   │
        │ Keep update  │          │ Revert + Queue│
        └──────────────┘          └──────────────┘
                                          │
                                          ▼
                                ┌──────────────────┐
                                │ Add to Queue     │
                                │ Show in UI       │
                                │ Retry later      │
                                └──────────────────┘
```

## 📱 Responsive Breakpoints

```
┌─────────────────────────────────────────────────────────────────┐
│  Screen Width                    UI Mode                         │
├─────────────────────────────────────────────────────────────────┤
│  < 768px (Mobile)                                                │
│  ├─ Admin                    → Desktop UI (forced)              │
│  └─ Volunteer                → Mobile UI                        │
│                                                                  │
│  ≥ 768px (Tablet/Desktop)                                        │
│  ├─ Admin                    → Desktop UI                       │
│  └─ Volunteer                → Desktop UI                       │
└─────────────────────────────────────────────────────────────────┘
```
